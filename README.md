### **1. Basic Singly Linked List Implementation**

### **2. Doubly Linked List Implementation**

This file shows how to implement:

* Nodes with both `next` and `prev` pointers
* Forward and backward traversal
* Insert and delete operations at various positions

Doubly linked lists allow bidirectional movement through the data.

---

### **3. Circular Singly Linked List Implementation**

A circular version of a singly linked list where:

* The last node points back to the head
* Useful for applications like round-robin scheduling

This file includes operations like insertion, deletion, and circular traversal.

---

### **4. Circular Doubly Linked List Implementation**

The most flexible list type in this repository:

* Nodes link both forward and backward
* The last node loops back to the head
* Excellent for implementing cyclic data structures
